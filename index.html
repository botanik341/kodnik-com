File protection and undo

Before Kodnik edits or fixes a file, it saves a restore point. After the work is done, you see a change notice under the code and can restore the file to its previous state with one click using Undo change.

Hardware

Kodnik adjusts itself to your hardware.

On first launch, Kodnik scans your computer, selects the compute backend and calculates how to split the model between your GPU and system memory. You install it and work — Kodnik handles the setup.

Detects your hardware

It detects NVIDIA, AMD or Intel graphics, reads exact VRAM, driver version, CPU core count and RAM amount before launching the model.

Selects the backend: CUDA or Vulkan

Based on your GPU, it selects the best compute backend — CUDA for NVIDIA, Vulkan for AMD and Intel — or CPU mode when no GPU is available.

Calculates model placement

It calculates how many model layers fit on the GPU and how many should stay on CPU and RAM, so it can use your hardware without overfilling memory.

Works on different GPUs

NVIDIA, AMD, Intel and CPU-only when needed. Kodnik adapts to your machine and does not require one specific GPU model.

Accurate GPU memory detection

It reads the real amount of GPU memory, including cards above 4 GB, so it can choose the correct runtime settings.

Reliable startup

If something cannot be detected, Kodnik falls back to a safe mode and still starts the model. The app is designed to start reliably.

FAQ

Questions before purchase.

What exactly does Kodnik do with my project?

You open a project folder and describe the task. Kodnik reads your files, shows a plan and then applies code changes itself — creates new files, edits existing ones and fixes errors. Before each change, it saves a restore point.

What computer do I need?

Windows 10/11 x64, CPU with AVX2, 32 GB RAM and about 24.8 GB of disk space for the app and model. A GPU is not required; with 6 GB VRAM work is more comfortable, and with 8 GB+ VRAM it is faster.

Do I need internet?

Internet is needed for first setup, ONLINE mode, model download and updates. Daily local work can be done offline.

Is my code safe?

Yes. In OFFLINE mode, everything stays on your computer. In ONLINE mode, Kodnik sends only the query needed to search for information — never the whole project folder.

Does it work without a GPU?

Yes, Kodnik works on CPU only. A GPU with at least 6 GB VRAM speeds up work, and 8 GB+ VRAM gives the best comfort.

Do I pay a subscription?

No. You pay once — no subscription and no cloud account. During the beta period, updates are included.

Will I get updates?

Yes. Kodnik has a built-in updater and tells you when a newer version is available. During the beta period, updates are included.

How long does installation take?

The app installation takes a few minutes. First setup and model download take longer, depending on your connection.
